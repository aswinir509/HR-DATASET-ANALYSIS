# HR Analytics Dashboard Using Power BI

# Abstract

Human Resource Analytics plays an important role in understanding employee performance, workforce distribution, employee attrition, salary trends, and organizational growth. The main objective of this project is to develop an interactive HR Analytics Dashboard using Power BI to visualize employee-related insights and support data-driven decision-making.

The dashboard was created using an HR dataset containing employee information such as job roles, department, education field, monthly income, daily rate, employee count, and performance-related attributes. Different visualization techniques such as bar charts, line charts, donut charts, tables, cards, and slicers were used to analyze and present the data effectively.

This project helps organizations identify workforce patterns, salary distribution, employee demographics, and departmental analysis. The dashboard improves reporting efficiency and assists HR managers in making strategic decisions.

# 1. Introduction

In modern organizations, Human Resource Management requires proper analysis of employee data to improve productivity and employee satisfaction. Traditional reporting methods are time-consuming and difficult to interpret. Business Intelligence tools such as Power BI provide advanced visualization capabilities that simplify data analysis.

This project focuses on building an HR Analytics Dashboard that enables organizations to monitor employee information efficiently. The dashboard provides insights into salary distribution, department performance, employee count, education field distribution, and job role analysis.

The project demonstrates how data analytics and visualization techniques can help HR departments make better decisions and improve workforce management.

The major objectives of this project are:

# 3. Problem Statement

Organizations often struggle to manage and analyze large amounts of employee data. Traditional spreadsheets and manual reports make it difficult to identify workforce trends and salary patterns.

The problem addressed in this project is the lack of an efficient system to visualize HR data interactively. The proposed solution is to create a Power BI dashboard that transforms raw HR data into meaningful insights for HR managers and organizational leaders.

# 8. Dashboard Design and Visualization

# 9. Analysis and Findings

The dashboard generated several important insights from the HR dataset.

## Key Findings

### Employee Distribution

* The organization contains approximately 1470 employees.
* Sales Executive roles have the highest employee count.
* Human Resources department has comparatively fewer employees.

### Salary Analysis

* Sales Executives contribute the highest monthly income.
* Managers and Research Directors also receive higher salaries.
* Laboratory Technicians and Sales Representatives have moderate salary ranges.

### Education Field Analysis

* Employees from Life Sciences represent the largest share.
* Medical and Marketing fields also contribute significantly.
* Human Resources and Technical Degree categories have smaller proportions.

### Department Analysis

* Research and Development (R&D) department has the highest contribution.
* Sales department contributes significantly to employee count and salary.
* HR department has the lowest distribution.

### Workforce Insights

* Employee distribution varies significantly across job roles.
* Income trends help identify organizational salary structures.
* Educational background analysis supports workforce planning.
